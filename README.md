# RedFeed

A minimalist Reddit video player built with React and Vite.

## Features
- Infinite scroll feed
- Native Reddit video playback
- Support for external embeds (YouTube, etc.)
- Dark mode optimized
- Subreddit switcher

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
